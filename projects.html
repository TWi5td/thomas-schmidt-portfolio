Mealy FSM GCD Calculator (Spring 2024)

Designed and implemented a Mealy Finite State Machine (FSM) in Verilog to compute the Greatest Common Divisor (GCD) of two 8-bit integers using the Euclidean algorithm on a Zybo FPGA. Verified functionality through behavioral and post-synthesis simulations in Vivado 2016.4, testing input pairs like (8,20)=4, (18,45)=9, and (28,49)=7. Tools: Vivado 2016.4, Verilog, Zybo FPGA.

GitHub Repository | Lab Report

4-bit LFSR with BRAM Interface (Spring 2024)

Designed a 4-bit Linear Feedback Shift Register (LFSR) interfaced with an 8-entry BRAM on a Zybo FPGA, generating and storing pseudo-random sequences. Managed multi-clock domains (125 MHz for BRAM, 10 MHz for LFSR) and implemented a circular buffer. Simulated in Vivado 2016.4, verifying storage and overwriting behavior over 11 cycles. Tools: Vivado 2016.4, Verilog, Zybo FPGA.

GitHub Repository | Lab Report

Transistor Characterization (VLSI Design, Spring 2025)
